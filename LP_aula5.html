1. Operador condicional (?:)

Ao invés do if/else podemos usar o operador condicional (?:).

O código à seguir lê uma nota e imprime "Aprovado" se a nota for maior que 6.0 ou "Reprovado" caso contrário: 

#include <stdio.h>

int main(void) {
  float nota;

  printf("Digite a nota: ");
  scanf("%f", &nota);

  if( nota >= 6.0 ) {
    puts("Aprovado!");
  } else {
    puts("Reprovado");
  }

  return 0;
}

Podemos usar o operador condicional e simplificar o código. 

#include <stdio.h>

int main(void) {
  float nota;

  printf("Digite a nota: ");
  scanf("%f", &nota);

  puts( (nota >= 6.0) ? "Aprovado" : "Reprovado" );

  return 0;
}

A sintaxe do operador ?: é: 

condição ? true : false

Para facilitar a leitura podemos usar parênteses.

1.1. Exercícios

  1. Escreva um programa em C que solicite ao usuário a idade de um paciente em um hospital geriátrico, usando o operador condicional (?:), imprima "Pode ser atendido" se a idade for maior ou igual a 60, ou "Não pode ser atendido" caso contrário.
  2. Crie um programa em C que peça ao usuário dois números e, utilizando o operador condicional (?:), imprima o maior dos dois números.

2. Blocos

Em C, um comando termina com ponto-e-vírgula (;). 

comando1;
comando2;

Um comando vazio pode ser criado assim: 

comando1;
;
comando2;

Cuidado ao usar ponto-e-vírgula, o if espera um comando: 

nota = 5.0;

if( nota >= 6.0 ) 
  puts("Aprovado!");

Se um ponto-e-vírgula aparecer temos um problema: 

nota = 5.0;

if( nota >= 6.0 );
  puts("Aprovado!");

O erro é sutíl, o que o computador realmente executa é: 

nota = 5.0;

if( nota >= 6.0 )
  ;

puts("Aprovado!");

Ou seja, sempre será impresso "Aprovado!" já que o comando de impressão ficou fora do if.

Onde temos um comando ou comando vazio podemos ter vários comandos também usando blocos de comandos que iniciam em { e terminam em }. Para o compilador, um bloco de comandos é o mesmo que um único comando. 

nota = 5.0;

if( nota >= 6.0 ) {
  puts("Aprovado");
  puts("Parabéns!");
}

3. Comando while

O comando de repetição (loop) while repete uma ação enquanto uma condição for true: 

while( condição ) 
  comando;

Para repetir vários comandos usamos o bloco: 

while( condição ) {
  comando1;
  comando2;
  comando3;
}

Vamos contar até 5 usando while: 

#include <stdio.h>

int main(void) {
  int contador = 1;

  while( contador <= 5 ) {
    printf("%d ", contador);
    contador = contador + 1;
  }

  return 0;
}

3.1. Exercícios

  1. Escreva um programa em C que use um loop while para imprimir os números de 1 a 10.

3.2. Funcionamento do while

O comando while repete os comandos enquanto o teste resultar em true. Por isso é importante que ajustemos os valores que serão usandos na condição e que o valor seja alterado até que encerre o while. Se a condição for sempre true a repetição não terminará e teremos o chamado loop infinito.

Sendo assim, o comando while deve ter: 

inicialização

while( condição ) {
  comandos;

  alteração;
}

Na inicialização, ajustamos as variáveis que serão usadas na condição. Se a variável usada em um teste não conter valor definido (lixo de memória) o comportamento é imprevisível. 

#include <stdio.h>

int main(void) {
  int contador;

  while( contador <= 5 ) {
    printf("%d ", contador);
    contador = contador + 1;
  }

  return 0;
}

A condição é obrigatória, se faltar teremos um erro de sintaxe e o programa não será compilado. 

#include <stdio.h>

int main(void) {
  int contador = 1;

  while( ) {
    printf("%d ", contador);
    contador = contador + 1;
  }

  return 0;
}

A alteração ou atualização modifica a variável de controle do loop, até que o teste retorne false e a repetição termine. Se não houver alteração, o loop se torna infinito: 

#include <stdio.h>

int main(void) {
  int contador = 1;

  while( contador <= 5 ) {
    printf("%d ", contador);
  }

  return 0;
}
